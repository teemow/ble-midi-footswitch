# ble-midi-footswitch

An ESP32 (Wemos Lolin32) based footswitch controller 

Material:

 * Aluminium box (1590DD)
 * Wemos Lolin32
 * Wemos LED Matrix 8x8
 * Wemos prototype shield
 * USB-B socket to micro USB
 * 2 MIDI sockets (5 pin)
 * 6 10mm LEDs
 * 6 10mm LED sockets
 * 6 momentary footswitches
